Site Identity

Basic information about this website.

Brand Colors

Your two main brand colors. Pick them here and click Apply, and they’ll be used across every page of your site. They start from the colors your site already uses, so you can adjust from there.

Heading Styles

Choose the font, size, weight, alignment, and color for your headings across the whole site. Headings are the titles on your pages — H1 is the main page title, H2 the section titles, and so on down to H6. Leave any box empty to keep what your design already uses. (A single page can still set its own heading look in that page's settings.)

Favicon

The small icon shown in browser tabs, bookmarks, and history. Use a square PNG, SVG, or ICO — at least 32×32px (512×512 recommended).

Analytics

Google Analytics 4, Tag Manager, and Ads tracking — with live data and monthly reports.

Loading…

Reports auto-generate for the previous month when you open this tab.

1 · Tracking IDs

Enter any combination — the snippet adapts. GA4 fires via gtag.js; GTM loads the container; the Google Ads tag piggybacks on gtag.

2 · Reporting API (live data & reports)

Dashboard and reports read from the GA4 Data API using a Google Cloud service account. One-time setup:

  1. Google Cloud Console → create/select a project → enable Google Analytics Data API
  2. IAM → Service Accounts → create one → Keys → Add key → JSON
  3. Paste the downloaded JSON below
  4. GA4 Admin → Property Access Management → add the service account email as Viewer
Checking…

3 · Custom code (header & body)

Paste any extra tracking or script snippets beyond GA4/GTM/Ads. Header code is injected into <head>; body code just before the closing </body>. They're grouped by purpose (to line up with a consent banner) but all load today. Saving publishes them to every page automatically.
